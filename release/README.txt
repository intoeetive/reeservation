This ZIP archive contains two versions of rEEservation module for ExpressionEngine.

2.4.5 is stable release and can be used on production sites.

2.5 is EXPERIMENTAL version and is not stable (use it on your own risk). 
It introduces support for pre-defined time ranges, which is useful for:
- event booking
- exact-time booking
- variable prices for different periods.
The docs for 2.5 version is in docs folder.
However 2.5 is discontinued in development and no support is offered.
